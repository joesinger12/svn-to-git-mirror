;  Comments:
;     MAX RECOMMENDED CTRL X  =  585
;     MAX RECOMMENDED CTRL Y  =  285
;     Max text file line Length  =  300 characters
;
;   * Not (yet) a feature of CBECC-Com13: --
;           Page Module Settings (compared via Bitwise OR to allow mode combinations)
;                 1 => Project-Mode Only     2 => Architecture Mode Only
;                 4 => Lighting Mode Only    8 => Mechanical Mode Only
;                15 => ALL Modes
;                (add selected individual mode values for other combinations)
;
;  Screen-Specific Help IDs = 100,000 + PageID
;
;-------------------------------------------------------------------------------
                                                                                                                                                                                                                                                
;-------------------------------------------------------------------------------
;
;  Format for Header = "TAB" => Defines a new Wizard Dialog or Dialog Tab:
;
;     1: Header "TAB"
;
;     2: Name of BDBase component for which data will be displayed and modified
;        (note: ALL data displayed/modified must be properties of this component)
;
;     3: Unique Integer Page ID - also used as Topic Help ID for context-sensitive
;        help.
;
;     4: Module(s) - not used for wizard dialogs - allows for different dialog
;        tabs to be presented based on the current UI module.
;
;     5: Caption presented in wizard dialog's outer frame or on the tab for tabbed
;        dialogs.
;
;-------------------------------------------------------------------------------
;
;  Format for Header = "CTRL" => Defines a new dialog child object
;
;     1: Header "CTRL"
;
;     2: Name of Property that this UI control displays/allows edit of
;        (this property must be a property of the component type specified above
;         for this dialog/tab)
;
;     3: Integer ID for this control - must be unique within this dialog for all
;        controls other than TDCT_Labels which display static text and do not
;        toggle on/off.
;
;     4: Control Type:
;           TDCT_Edit   : Edit control allowing the display and/or entry of strings or numbers
;                          - provides access to BDBase properties of type BDP_Int (integer),
;                            BDP_Flt (float) or BDP_Str (can also display but not edit
;                            data of type BDP_Sym (symbol) and BDP_Obj (object/component).
;           TDCT_Combo  : Combobox control (always CBS_DROPDOWNLIST | CBS_AUTOHSCROLL)
;                          - provides access to BDBase properties of type BDP_Sym (symbol)
;                            or BDP_Obj (object/component) only.
;           TDCT_Button : Push button control
;                          - If "Val" > 0, then WM_BUTTONPRESSED sent to CMainFrame to process
;           TDCT_Check  : Checkbox control
;                          - provides access to BDBase properties of type BDP_Int (integer)
;                            only, where 0 => unchecked and 1 => checked.
;           TDCT_RadioG : The first of a group of radio buttons
;                          - provides access to BDBase properties of type BDP_Int (integer) only.
;           TDCT_Radio  : The second - Nth radio buttons in a group
;                          - <same as TDCT_RadioG>
;    NYI    TDCT_WMF    : Displays a windows metafile
;                          - connected to BDBase property of type BDP_Str (string), where the
;                            strings contents is the filename of the windows metafile to be
;                            displayed (the file must exist in the screen graphics directory)
;    NYI    TDCT_Date   : Displays like a combo box, but drops down to provide a calendar
;                         date selection control
;                          - provides access to BDBase properties of type BDP_Int (integer)
;                          - minimum and maximum selectable calendar dates can be set by
;                            connecting this control to an array of BDP_Int property, where
;                            the connected property is the date displayed/edited and the
;                            subsequent array elements contain the minimum and maximum dates
;           TDCT_Label  : Displays either static text labels or a combination of static and
;                         BDBase data
;                          - displays all types of BDBase data
;           TDCT_Line   : Displays a line on the screen
;           TDCT_List   : Displays a list control - used for selection of "active" component
;     NYI   TDCT_ChkLst : Displays a check-list box control
;     NYI   TDCT_RTF    : RTF control (property = filename in relation to screens directory to display)
;     NYI   TDCT_HlpBtn : Small iconic help button
;     NYI   TDCT_IcnBtn : Other miscellaneous small iconic push buttons
;
;     5: X coordinate where control is located
;           - For TDCT_Label controls, this can represent either the left, right or center X
;             coordinate based on the value of "Control Label Justif." field (
;           - For TDCT_Line controls, this is the "From X" coordinate
;           - For ALL OTHER controls, this is the X coordinate of the top left corner of the control
;           - added note for TDCT_IcnBtn : negative values result in positioning starting from the right
;             edge of the parent dialog and moving left (SAC 4/8/09)
;
;     6: Y coordinate where control is located
;           - For TDCT_Line controls, this is the "From Y" coordinate
;           - For ALL OTHER controls, this is the Y coordinate of the top of the control
;
;     7: Ht is the height of the control
;           - For TDCT_Line controls, this is the "To X" coordinate
;           - For TDCT_Combo controls, this is the maximum pixel height including the dropped list
;           - For ALL OTHER controls, this is the pixel height of the control
;
;     8: Wd is the width of the control
;           - For TDCT_Line controls, this is the "To Y" coordinate
;           - For TDCT_Label controls, a value of 0 will be ignored, while values > 0 will cause
;             the label string to be truncated in the event its pixel width exceeds the specified value
;           - For ALL OTHER controls, this is the pixel width of the control
;
;     9: Val - or Special Value is interpretted in a variety of ways based on the control type:
;           TDCT_Edit &
;           TDCT_Label  : When displaying string data:
;                              = -1  =>  limit # chars to max path length (255) and ensure chars allowed in file path
;                              <  0  =>  no limit on # of characters
;                              >= 0  =>  limit on # of characters
;                         When displaying numeric data:
;                              = 11  =>  use decimal precision of value in database, w/ commas inserted for values > 999
;                              >= 0  =>  decimal precision, with commas inserted for values > 999
;                              = -1  =>  integer value to be displayed as a date in long form ("%H:%M, %a, %b %d, %Y")
;                              = -2  =>  integer value to be displayed as a date in short form ("%m/%d/%y")
;                              = -3  =>  integer value to be displayed as a date in ISO format (i.e. 2018-06-12T17:22:34-07:00)
;                              < -3  =>  0 decimal precision and no commas inserted for values > 999 (year such as 1998)
;           TDCT_Combo  : Val % 100  =>  Height of combobox when "closed"
;                         Val / 100  =>  Flag values providing the following functionality:
;                                               1  :  Prevent inclusion of "- none -"
;                                               2  :  Prevent inclusion of "- Create -"
;                                               4  :  Start with "- baseline run -" in place of "- none -" (but serves same purpose)
;           TDCT_Button &
;           TDCT_IcnBtn : =	999   => Access to the tabbed dialog associated w/ the object assigned via the specified Object:Property (DBID)
;										1001-1300 (?)	=>  Delete CartesianPt object from PolyLp
;										2000-2300 (?)	=>  Create a new CartesianPt object and insert into PolyLp
;										3001	=>	Access PolyLp child of Spc
;										3002	=>	Access PolyLp child of Ceiling   
;										3003	=>	Access PolyLp child of ExtFlr    
;										3004	=>	Access PolyLp child of ExtWall   
;										3005	=>	Access PolyLp child of IntFlr    
;										3006	=>	Access PolyLp child of IntWall   
;										3007	=>	Access PolyLp child of Roof      
;										3008	=>	Access PolyLp child of UndgrFlr  
;										3009	=>	Access PolyLp child of UndgrWall 
;										3010	=>	Access PolyLp child of Window    
;										3011	=>	Access PolyLp child of Skylight  
;										3012	=>	Access PolyLp child of Door      
;										3013	=>	Access PolyLp child of ExtShdgObj
;										3050	=>	Access to Win Shades diagram dialog
;										3051	=>	Browse (file-open) dialog to select ExcptDsgnModelFile
;           TDCT_Check  : Unused
;           TDCT_RadioG &
;           TDCT_Radio  : The integer value corresponding to the checked radio button state
;           TDCT_WMF    : Unused
;           TDCT_Date   :  =   0  =>  BDBase data is a full (time_t) value
;                          =   1  =>  BDBase data is a 3-4 digit integer = (Month * 100) + Day
;           TDCT_Line   :  =  -1  =>  Windows 3-D shadow line (shadow color above, highlight color below)
;                          =  -2  =>  ButtonShadow color, single pixel width
;                          >   0  =>  Black line width
;           TDCT_List   :  =   0  =>  List ALL components of the type specified
;                          =   1  =>  List only children of the active PARENT component of the type specified
;           TDCT_ChkLst :  <= 10  =>  Y Margin for displaying entries in listbox (typically between 1-5)
;                          >  10  =>  ID of RTF control that gets refreshed upon selection of each item in listbox
;           TDCT_HlpBtn : The number of Help Item definitions that follow this control's definition
;
;     10: Font - applies to font both inside control and the control's label
;                                Pixel Ht
;                                @ Sm Sys
;                                Resources  Weight  Orient  Ital  Undrln  Font Name
;                                ----------------------------------------------------
;                 FNT_STD           16         0     Horiz    0      0    Arial
;                 FNT_BOLD          16       650     Horiz    0      0    Arial
;                 FNT_UNITS         13         0     Horiz    0      0    Arial
;                 FNT_MENU          36       700     Horiz    0      0    Arial Bold
;                 FNT_TOOLBAR       15         0     Horiz    0      0    Arial
;                 FNT_VSMALL        15         0     Vert     0      0    Arial
;                 FNT_DATATYPE      12         0     Horiz    0      0    Arial
;                 FNT_VDATATYPE     12         0     Vert     0      0    Arial
;                 FNT_UNDERLINE     16         0     Horiz    0      1    Arial
;                 FNT_ITALIC        16         0     Horiz    1      0    Arial Italic
;                 FNT_ITALUNDER     16         0     Horiz    1      1    Arial Italic
;                 FNT_VD_STD        16         0     Horiz    0      0    Verdana
;                 FNT_VD_BOLD       16       650     Horiz    0      0    Verdana
;                 FNT_VD_STDU       16         0     Horiz    0      1    Verdana
;                 FNT_SYMBOL        16         0     Horiz    0      0    Wingdings
;
;     11: Control Label - text string used to label the UI control
;           TDCT_Label  : When displaying these controls, the final displayed string is always
;                         this label string + the BDBase data value (only if a component Property
;                         Name is provided) + any string contained in the Status Bar Message column
;           TDCT_ChkLst : Name of wizard rulelist to be evaluated following each listbox selection
;
;     12: Control Label Justification - options include: TA_LEFT, TA_RIGHT or TA_CENTER
;
;     13: Control Label DX - the X coordinate OFFSET in relation to the controls X position to
;         display the label at - EXCEPT for the following special cases:
;           TDCT_Label  : Pixel width of area to be erased between each display of modified data
;           TDCT_Button &
;           TDCT_IcnBtn : when Val = 91 (Navigation to detailed EEM wizard screens) then this Label DX
;                               value is the corresponding 1-based EEM component index to be edited
;                         ELSE, when Val = 104 or 105 and this value is = 1 => CREATE New ShellWiz or
;                               HVACWiz component and then edit that component (DD Wizard)
;
;     14: Control Label DY - the Y coordinate OFFSET in relation to the controls Y position to
;         display the label at.
;
;
;     15: Units Label Display Flag - 1 to display units label or 0 to not
;           TDCT_ChkLst :  = 0  =>  only associated RTF control is refreshed following active item change
;                          = 1  =>  entier screen is refreshed following active item change
;
;     16: Units Label Justification - options include: TA_LEFT, TA_RIGHT or TA_CENTER
;
;     17: Units Label DX - the X coordinate OFFSET in relation to the controls RIGHT X position to
;         display the label at.
;           TDCT_Label  : Units labels cannot be automatically displayed via Label controls
;                         = -1 => Don't erase label via writing w/ gray text (when display condition = FALSE)
;                         > 0 & Label is LEFT justified  => Draw shadowline from ~8 pixels to the right
;                                                           of the label to th UnitsDX coordinate
;           TDCT_List   : > 0  =>  Navigation ID, similar to the navigation for Button controls
;           TDCT_IcnBtn : ID of icon to be displayed.  Valid values include:
;                           1  =>  Diskette (save)
;                           2  =>  EEM Wizard (user in whole site/bldg EEM wizard mode)  - SAC 4/7/09
;
;     18: Units Label DY - the Y coordinate OFFSET in relation to the controls Y position to
;         display the label at.
;           TDCT_IcnBtn : > 0 => button drawn w/ flat focus rectangle
;
;     19: Condition(s) under which this control/label/etc is editable
;           None      => No condition applies => always make control editable
;                        (subject to ruleset DataType settings)
;           Otherwise => 1-5 conditions (separated by operators " Or " or " And ")
;                        (all operators must be consistent, a mixture of "Or"s and "And"s are not allowed)
;                        where each condition is specified in the form:
;                          <property> <operator> <numeric value or another property>
;                             where <property> is a property of the local component, and
;                             <operator> is:  =, <=, >=, <, >, or !=
;
;     20: Condition(s) under which this control/label/etc is displayed
;           None      => No condition applies => always display the control
;                        (subject to ruleset DataType settings)
;           Otherwise => 1-5 conditions (separated by operators " Or " or " And ")
;                        (all operators must be consistent, a mixture of "Or"s and "And"s are not allowed)
;                        where each condition is specified in the form:
;                          <property> <operator> <numeric value or another property>
;                             where <property> is a property of the local component, and
;                             <operator> is:  =, <=, >=, <, >, or !=
;
;     21: Condition Left Erase DX - X Pixel Offset in relation to control's Left X edge
;         to erase in the event the control is being removed (condition failed)
;
;     22: Condition Right Erase DX - X Pixel Offset in relation to control's Right X edge
;         to erase in the event the control is being removed (condition failed)
;
;     23: Status Bar / Tool Tip Message - this message will override any message contained
;         in the separate ToolTips.txt file => if no status bar message is defined here, then
;         the one defined in the ToolTips file will be used
;           TDCT_Spread  : (SAC 9/10/01 added) If specified and translates to object assignment [Comp:]Property,
;                          then the referenced component will be the one edited/viewed within the spreadsheet
;
;  For Spreadsheet Controls Only, the above data is followed by:
;     First line following CTRL definition:
;        - Name of Property which indicates a maximum number of rows of data which may be displayed in the spreadsheet (or "None"),
;        - Character string to be displayed in the top-leftmost header cell (above the row headers) - SAC 8/9/02 added
;        - Row Header Font,
;        - Row Hdr Column Width,
;        - Column Hdr Row Height,
;        - Row Grouping, (default = 1)  - SAC 2/1/01 Added to enable modified row labels and lines to separate row groups
;        - Number of Locked Columns,
;        - Total Number of Columns,
;     Following N lines describing each column in the spreadsheet:
;        - Column Title,
;        - Column Width,
;        - Misc Value (decimal precision for numerics, max # characters for strings),
;        - 0/1 Flag to Append Units String to Column Label,
;        - 0/1 Flag to allow this column to be edited,
;        - Property Name,
;        - Item Help ID (0 to use BDBase default),
;        - Condition which must evaluate to TRUE in order for cells of this column to be editable (or None if no condition applies),
;        - Condition which must evaluate to TRUE in order for cells of this column to be populated with data (or None if no condition applies),
;        - Tool Tip Message
;
;  For Spreadsheet Controls Only, the above data is followed by:
;     Following N lines each define a single "Help Item" (where N = control's "Val" field), including
;        - Help System ID,
;        - Menu Text (displayed in menu when multiple quick help items available),
;        - Condition which must evaluate to TRUE for access to this item to be made available (or None if no condition applies),
